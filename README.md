# Python String Processing

A collection of Python programs for common string manipulation and text processing tasks.

## Programs

### `punctuation.py`
Takes a paragraph as input, strips all punctuation marks (`.,!?;:`), then counts and displays the number of unique words.

### `references.py`
Reformats academic references: converts author names to title case and ensures only the first letter of the title is capitalised, leaving the rest of the reference unchanged.

### `search_word.py`
Takes a sentence and a search word as input, performs a case-insensitive search, and returns whether the word exists and the index of its first occurrence.

## How to Run

```bash
python punctuation.py
python references.py
python search_word.py
```
